* Skip-gram三个主要创新：
第一，将常用词对或短语视为模型中的单个“词”；第二，对频繁的词进行子采样以减少训练样例的数量；第三，使用“负抽样”技术修改优化目标，该技术使每个训练样本仅更新模型权重的一小部分。
